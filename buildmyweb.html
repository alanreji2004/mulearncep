Build My Web

"Build-my-web" was an intensive one-month web development bootcamp that integrated online and offline modes, offering participants a comprehensive learning journey in website creation using HTML, CSS, Bootstrap, and Javascript.


Throughout the program, participants delved into the fundamental building blocks of web development. They started with HTML, learning how to structure web content effectively. CSS was explored extensively to empower participants with the skills to design and style web pages for a visually appealing user experience. A significant focus of the bootcamp was on Bootstrap, a widely-used front-end framework. Participants gained proficiency in leveraging Bootstrap to create responsive and mobile-friendly websites efficiently. Additionally, the program covered Javascript, enabling participants to add interactivity and dynamic features to their web projects. The bootcamp adopted a blended learning approach, combining online resources with offline sessions such as workshops and coding labs. This not only provided flexibility in learning but also fostered a collaborative and interactive environment. Expert instructors, who were seasoned professionals in the field, guided participants throughout the program, offering insights and mentorship.
